## Short.ly

a url shortener, a feather-weight bit.ly clone.

### Functionalities:

 * url valid check
 * url shorten in base62
 * store long to short
 * store short to long
 
### Design

 * Tool / Framework
  * node.js - backend server
  * mongoose - schema modeling
  * mongo - database
  * react - ui component
  * gulp - task manager and build system
  * browserify - bundle module
